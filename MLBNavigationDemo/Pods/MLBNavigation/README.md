## Installation 安装

下载demo，将MLBNavigation拖入项目中即可，或者cocopods安装。pod search MLBNavigation 搜索不到，请先更新pod setup

```ruby
pod 'MLBNavigation'
```

## Usage method 使用方法

创建UIViewController的时候，直接继承MLBBaseController.h即可，详情可看demo
